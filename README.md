# 📚 Online Attendance Management System

## 🚀 Project Overview
This is a full-stack web application designed to manage student attendance digitally. It allows teachers/admins to mark attendance, view reports, and monitor student presence efficiently.

---

## 🎯 Objectives
- Replace manual attendance with a digital system  
- Provide real-time attendance updates  
- Store data securely using a database  
- Generate attendance reports easily  

---

## ⭐ Features
- 🔐 User Login (JWT Authentication)
- 📝 Mark Attendance (Present/Absent)
- 📊 Dashboard (Total, Present, Absent)
- 📄 Attendance Reports
- 🔄 Real-time updates using Socket.io
- ☁️ Cloud database integration

## ⚙️ How It Works
1. User logs in using credentials  
2. Frontend sends request to backend  
3. Backend verifies user and returns token  
4. Attendance is marked and saved in MongoDB  
5. Dashboard calculates present/absent  
6. Reports display stored attendance
